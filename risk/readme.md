# ⚠️ Risk

This folder focuses on **real risk**, not perceived risk.

Real risk is:
- Permanent loss
- Forced selling
- Fragility
- Hidden leverage

Not:
- Daily volatility
- Temporary drawdowns

---

## Key Ideas
- Risk hides during calm periods
- Volatility is not danger
- Survival enables compounding

---

## Use This Folder When:
- Markets feel “safe”
- Returns feel easy
- Leverage seems tempting

---

> If you survive, you can recover.
> If you don’t, returns don’t matter.
